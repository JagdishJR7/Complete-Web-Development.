Loops in JS ::

  • For loop : To iterate array element , we can use for loop.

  • ForEach loop : it is more easy way to use for loop than simple for loop.

  • For of loop : it is also a another way to use for loop which is faster and has less code.

  • for in loop : to iterate objects in JS, means to iterate the keys in array or object.

  • while loop : this is same as for.

  • Do while loop : it runs for atleast once.
